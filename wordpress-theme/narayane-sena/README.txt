Professional membership and referral management theme for WordPress.

== Description ==

Narayane Sena is a comprehensive membership and referral management platform built as a WordPress theme. Perfect for businesses looking to create a referral-based membership system with admin controls.

= Features =

* User Registration with Admin Approval
* Membership Management System
  - ₹5000 regular membership
  - ₹2000 with "SAVE3K" coupon code
  - Payment via QR code upload
  - Admin approval workflow

* Referral Program
  - Earn ₹2000 per successful referral
  - Track daily, weekly, and total earnings
  - View referral history
  - Unique referral codes for each user

* Admin Panel
  - Comprehensive user management
  - Membership approval workflow
  - Payment QR code management
  - Platform statistics and analytics
  - Earnings management

* User Dashboard
  - Profile management
  - Earnings tracking
  - Referral link sharing
  - Membership status

== Installation ==

1. Upload the 'narayane-sena' folder to the '/wp-content/themes/' directory
2. Activate the theme through the 'Appearance' menu in WordPress
3. The theme will automatically create necessary database tables
4. Use your existing WordPress admin account to manage the platform
5. Navigate to the Narayane Sena menu in admin panel to configure settings

== Frequently Asked Questions ==

= How do I set up payment QR codes? =

Navigate to the WordPress admin panel and look for the "Narayane Sena" menu. You can upload payment QR codes there.

= How do users purchase memberships? =

Users can register, then navigate to the Membership Purchase page. They scan the QR code, make payment, upload screenshot, and wait for admin approval.

= How does the referral system work? =

Each user gets a unique referral code. When someone registers using their referral link and their membership gets approved, the referrer earns ₹2000.

= Can I change the membership prices? =

Yes, you can modify the prices in the theme's functions.php file by editing the ns_get_membership_price() function.
